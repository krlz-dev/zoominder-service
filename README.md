# Zoominder Service

A Spring Boot REST API for pet care management.

## Features

- User management
- Pet profiles
- Medication tracking

## Tech Stack

- Java 17+
- Spring Boot
- H2 / PostgreSQL
- Flyway migrations

## Running

```bash
mvn spring-boot:run
```

## API Docs

Swagger UI available at `/swagger-ui.html` when running.
